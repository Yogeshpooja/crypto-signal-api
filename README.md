# Crypto Signal API

A scalable REST API with Authentication & Role-Based Access Control for crypto trading signals.

## Features
- User Registration & Login with JWT
- Role-Based Access (Admin/User)
- CRUD operations for Trading Signals
- PostgreSQL Database
- Responsive Frontend Dashboard

## Tech Stack
- Node.js, Express
- PostgreSQL
- JWT, bcrypt
- HTML/CSS/JS

## Setup Instructions
```bash
npm install
npm run dev
